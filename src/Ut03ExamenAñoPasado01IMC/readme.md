# IMC

Vamos a representar una persona a través de una clase, pero va a ser para un sistema de gestión de pacientes, con lo que esa clase va a tener diferentes atributos: nombre, apellidos, edad, altura, peso.

Además, hay que crear un método para calcular el IMC, cuya fórmula es IMC = peso / (altura * altura).

Según el IMC, Determina la categoría según el IMC:

Menos de 18.5: Bajo peso

18.5 a 24.9: Peso normal

25 a 29.9: Sobrepeso

30 o más: Obesidad

Modela la clase según tu criterio. Piensa qué constructor o constructores necesitas. No olvides incluir un método para mostrar la clase como un String.
