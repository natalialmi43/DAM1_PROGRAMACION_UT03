# Numeros primos
Crea una clase para representar números primos, que se llame NumeroPrimo, con un atributo que represente el número y dos métodos:

esPrimo(): devolverá si el número es realmente primo o no. Este método deberás utilizarlo en el resto de métodos.

siguiente(): devolverá otro NumeroPrimo que será el siguiente.

posicion(): devolverá la posición del número en el listado de números primos, tomando el 2 como el primer número primo.

sumaAnteriores(): devolverá la suma de todos los primos anteriores al número.

Hay que crear una clase con un main para crear un número primo y poder operar con él (un menú). Tienes que tener en cuenta que el usuario puede introducir cualquier número (sin decimales).
