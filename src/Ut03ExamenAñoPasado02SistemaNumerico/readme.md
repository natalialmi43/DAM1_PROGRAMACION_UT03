# Sistemas númericos

Crea una clase para representar un número entero. 

Debe tener dos constructores, uno que reciba un entero (decimal), y otro que reciba un String con el número y un entero con la base. Así se podrá utilizar el método Integer.parseInt(valor, base) Debe tener cuatro funciones que devuelvan un String con el valor de ese número en las diferentes bases (decimal, binario, octal y hexadecimal). Utiliza los métodos auxiliares de Integer (que son toBinaryString, toOctalString y toHexString).
